# 🧮 Calculadora de IMC

Aplicação web interativa para cálculo de IMC, desenvolvida com HTML, CSS e JavaScript, focada em lógica de programação, interatividade e experiência do usuário.

## 📌 Funcionalidades

- Cálculo automático do IMC  
- Classificação do resultado (abaixo do peso, normal, sobrepeso, etc.)  
- Interface simples e intuitiva  
- Validação de dados de entrada  

---

## 🧠 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, incluindo:

- Estruturação com HTML  
- Estilização com CSS  
- Lógica e interatividade com JavaScript  

---

## ▶️ Como usar

1. Insira seu peso (kg)  
2. Insira sua altura (m)  
3. Clique em calcular  
4. Veja seu resultado e classificação
